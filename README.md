# My Shell 🖥️

A custom Unix-like shell built in C language with support for pipes and redirection.

## Features
- ✅ Execute any system command (ls, echo, dir, whoami)
- ✅ Built-in commands (cd, exit, help)
- ✅ Output redirection (echo hello > file.txt)
- ✅ Input redirection (sort < file.txt)
- ✅ Pipes (ls | findstr shell)
